Plugins — atomic capabilities

A single API capability wrapped in a SwarmSpace manifest. One input, one output, one purpose.

  • Submit a JSON manifest for review

  • Community plugins: always free, no credit charging
  • Verified plugins: developer sets credit price per call

Workflows — composed chains

An ordered chain of plugin calls. One user intent, multiple API calls, one synthesised result.

  • Same manifest spec as plugins, plus a steps array
  • Community workflows: free APIs only
  • Verified workflows: any combination of free and paid
  • Revenue compounds: 3-plugin workflow earns on 4 transactions
  • Developers who own underlying plugins earn on both layers
  • At 1,000 runs/mo at 30cr ($3.00): $2,400/mo at 80% share

Common questions

Can I use SwarmSpace without LUMARA?
Yes. SwarmSpace Pro gives you full API access and plugin discovery without the LUMARA companion app. LUMARA Premium bundles both for $5 more per month.
Does Verified status cost money?
No. Verified is earned through performance, not purchased. Rankings are based on usage, upvotes, retention, developer activity, and error rate — reviewed monthly. No single signal can be gamed.
How does Developer Pro work?
Maintain at least one published Community or Verified plugin or workflow in good standing and Developer Pro is free. You get full SwarmSpace Pro access — all premium APIs — to test your own integrations. If you become inactive for 90 days, the plan reverts to standard pricing.
Do credits expire?
Credits expire 12 months from purchase date. No rollover at launch — partial carry-forward (up to 20%) is under review for post-launch based on user feedback.
What is the Founding Developer Programme?
The first 100 developers to achieve Verified status receive 85% revenue share permanently (vs the standard 80%), plus a Founding Developer badge on their public profile. The badge is scarce and non-transferable.
What infrastructure does SwarmSpace run on?
Plugin execution runs on Cloudflare Workers at the edge. Authentication via Firebase. The index, trust layer, and discovery API are hosted separately from plugin execution — developers' plugins run at their own endpoints. SwarmSpace never proxies sensitive user data.

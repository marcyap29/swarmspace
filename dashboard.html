How to use the API

Authenticate with a Firebase ID token (Bearer). Your tier determines which plugins you can call. Example request/response and code:

POST https://us-central1-arc-epi.cloudfunctions.net/swarmspaceRouter Authorization: Bearer <YOUR_FIREBASE_ID_TOKEN> Body (data wrapper for callable): { "data": { "plugin_id": "brave-search", "params": { "query": "What are the latest developments in AI?", "count": 8 } } }
{ "result": { "quota": { "limit": 100, "used": 42, "remaining": 58 }, "results": [ { "title": "...", "url": "...", "description": "..." } ] } }
curl -X POST https://us-central1-arc-epi.cloudfunctions.net/swarmspaceRouter \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_FIREBASE_ID_TOKEN" \ -d '{"data":{"plugin_id":"brave-search","params":{"query":"Latest AI news","count":5}}}'
const token = "YOUR_FIREBASE_ID_TOKEN"; const res = await fetch("https://us-central1-arc-epi.cloudfunctions.net/swarmspaceRouter", { method: "POST", headers: { "Content-Type": "application/json", "Authorization": `Bearer ${token}` }, body: JSON.stringify({ data: { plugin_id: "brave-search", params: { query: "Latest AI news", count: 5 } } }) }); const json = await res.json();

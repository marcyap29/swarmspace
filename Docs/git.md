---

## Commit Conventions

- **Format:** `type(scope): description`
- **Types:** `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`
- **Examples:**
  - `feat(auth): add OAuth sign-in`
  - `docs: update README file structure`
  - `fix(stripe): handle webhook signature`
